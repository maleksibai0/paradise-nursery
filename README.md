# Paradise Nursery Shopping Application

## Description
Paradise Nursery Shopping Application is a comprehensive platform designed for users to explore and purchase various plants, gardening tools, and related products. The application offers an intuitive user interface for a seamless shopping experience.

## Features
- User-friendly interface for easy navigation.
- Secure user authentication and account management.
- Extensive catalog of plants and gardening tools.
- Shopping cart functionality for easy management of purchases.
- Integration with payment gateways for secure transactions.
- Admin panel for inventory management.

## Technologies Used
- Frontend: React, Bootstrap
- Backend: Node.js, Express
- Database: MongoDB
- Authentication: JWT
- Deployment: Heroku

## Installation Instructions
1. Clone the repository:
   ```bash
   git clone https://github.com/maleksibai0/paradise-nursery.git
   ```
2. Navigate to the project directory:
   ```bash
   cd paradise-nursery
   ```
3. Install the dependencies:
   ```bash
   npm install
   ```
4. Set up environment variables as per the `.env.example` file.
5. Run the application:
   ```bash
   npm start
   ```

## Usage Guide
- Once the application is running, navigate to `http://localhost:3000` in your web browser.
- Create an account or log in to explore the features of the application.
- Browse the catalog and add items to your cart for purchase.
- Follow the checkout process to complete your orders.
